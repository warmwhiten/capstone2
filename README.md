# Usage
## 클라이언트 실행

node.js 설치가 필요합니다.

```bash
$ cd client
$ npm install
$ npm start
```

## 서버 실행

python3 설치가 필요합니다.

```
$ server/venv/Scripts/activate.bat
$ python app.py
```
